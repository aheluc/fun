# Fun
Fun是一个**基于Python的脚本语言**。
擅长处理数据流，其高度的随意性和动态性可以使其代码精简而灵活。

## 特点

### 自动lambda化
当一个表达式中的存在变量未定义时，则整个表达式变为lambda表达式。
`fun = x * y;`
实质上等价于
```
fun = {
    <- x * y;
};
```

### 定义函数不需要声明参数列表
*就像这样：*
```
fun = {
    x = arg;
    <- x;
};
```
只需要在调用时，给够所需的参数
`["arg": 5] -> fun;`

### 表是list与dict的融合
你可以这样定义一个表：`table = [1, 2, 3, "word": "hi!"];`

### 生成器可以展开为表
你也可以这样定义一个表：`table = [..range << [0, 5]];`
